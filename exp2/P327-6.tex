\subsection{P327-6}
用Euler法和预估-校正方法求解初值问题
\begin{equation*}
\begin{cases}
y' & = x + y \\
y(0) & = 0, x \in [0, 1]
\end{cases}
\end{equation*}
$h=0.1$，精确到小数点后5位，并与精确解$y = -x - 1 + 2 e^x$相比较
\begin{SOLVE}

\begin{lstlisting}

>> [x, y2] = m_eulert(@(x, y) x + y, [0, 1], 1, 0.1);
>> y3 = -x - 1 + 2 * exp(x);
>> [y1;y2;y3]
